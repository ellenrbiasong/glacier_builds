# ❄️ GlacierOS
A lightweight Arch-based Linux distribution featuring the beautiful Nord color scheme. Cold. Clean. Fast.
## Features
- 🎨 Nord-themed desktop (arctic blues, cool grays)
- ⚡ Minimal & fast (under 1GB)
- 🔄 Rolling release (Arch Linux base)
- 🖥️ KDE Plasma desktop
## what's inside
- Base: Arch Linux
- Desktop: KDE Plasma
- Login: SDDM
- Theme: Nord
